# Diario de Entrenamiento 💪

PWA para llevar el seguimiento de ejercicios y pesos del gimnasio.

## Subir a GitHub Pages

1. Crea un repo en GitHub llamado `gym-tracker` (o el nombre que quieras)
2. Sube todos estos archivos al repo
3. Ve a **Settings → Pages → Source** y selecciona **GitHub Actions**
4. En el archivo `vite.config.js`, cambia la línea `base: '/gym-tracker/'` por el nombre de tu repo si es diferente
5. Haz push a `main` — el workflow se encarga del resto

La app estará en: `https://TU_USUARIO.github.io/gym-tracker/`

## Instalar en el móvil (Android)

1. Abre la URL en Chrome
2. Toca el menú (⋮) → **Añadir a pantalla de inicio**
3. Ya tienes el icono como si fuera una app nativa

## Instalar en iPhone

1. Abre la URL en Safari
2. Toca el botón compartir → **Añadir a pantalla de inicio**
